Для запуска скрипта нужно в index.html в поля ввести данные:
 - Страничка которую нужно выкачать
 - Предполагаемый root (разные сайты могу по разному указывать URL ссылки на скрипты и картинки)
Нажать кнопку "Готово" и браузер откроет две вкладки:
1. Первая нужна для того чтобы скрипт считал DOM дерево файла и выгрузил все ссылки (img, script, link) с него.
Скрипт копирует файлы в свой каталог в директорию "uploads". С каждым новым использываением скрипта содержимое папки перезаписывается
2. Вторая вкладка открывается автоматически, собственно она отображает скаченую страничку. Если не все елементы отображаются коректно,
нужно перезагрузить эту вкладку (Ctrl+R) чтобы пути перенастроились. Если этот способ не помог:
В каталоге "uploads" должен появится файл "error.txt" в котором содержатся все ссылки которые не получилось выкачать. 
Такие ссылки можна вручную скачать, это в основном CDN ссылки на библотеки которые не можно скачать данным способом. 

Сама структура файлов это:
 - index.html - Интерфейс для роботы со скриптом
 - index.php  - Файл для открытия сайта и запуска скриптов.
 - save.php   - Ядро скрипта. Вся обработка находится тут.
